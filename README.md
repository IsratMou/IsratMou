## ✨ Research Highlights

<details open>
<summary>Show research snapshot</summary>

**MOI-Lite + E-SATF — Hierarchical IoT IDS (TON-IoT Benchmark)**

| Metric | Result |
|---|---|
| Stage-1 Binary F1 | **99.79%** |
| Stage-2 Multiclass Macro F1 | **94.37%** |
| Parameters | **37K** (−36.5% vs DNN/CNN) |
| Size after INT8 Quantization | **65 KB** |
| Adversarial Robustness (FGSM) | **+26% relative gain** |
| Statistical Equivalence | McNemar p > 0.05 ✅ |

> 💡 **Key idea:** A lightweight model that matches large baselines in accuracy, SHAP explanation stability, and adversarial robustness — while fitting inside an Arduino-class MCU.
